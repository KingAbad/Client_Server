IF USING WHILE CLIENT AND SERVER ON THE DIFFERENT NETWORKS THEN:

NOTE: ALWAYS RUN THE SERVER FIRST!


FOR SERVER SIDE

COMPILE USING	: gcc server.c -o server

RUN THE FILE	: ./server PORT_NUMBER
FOR EXAMPLE	: ./server 1234


FOR CLIENT SIDE

COMPILE USING	: gcc client.c -o client

RUN THE FILE	: ./client.exe IP_ADRESS_OF_SERVER PORT_NUMBER
FOR EXAMPLE	: ./client.exe 192.123.1.12 1234



IF USING WHILE CLIENT AND SERVER ON THE SAME NETWORK THEN:

NOTE: ALWAYS RUN THE SERVER FIRST!


FOR SERVER SIDE

COMPILE USING	: gcc server.c -o server

RUN THE FILE	: ./server PORT_NUMBER
FOR EXAMPLE	: ./server 1234


FOR CLIENT SIDE

COMPILE USING	: gcc client.c -o client

RUN THE FILE	: ./client.exe 127.0.0.1 PORT_NUMBER
FOR EXAMPLE	: ./client.exe 127.0.0.1 1234
